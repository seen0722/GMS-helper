# GMS-Helper 系統展示影片腳本 (無旁白/字幕版 + Redmine 整合)

**目標受眾:** 高階主管 (Executives)
**目標:** 展示 GMS-helper 系統的效率、AI 能力以及與 Redmine 的無縫整合。
**形式:** 螢幕錄影 + 背景輕音樂 + **淡入淡出文字 (On-Screen Text)**
**預計時長:** 2 - 2.5 分鐘

---

## 🎬 錄影設定建議
*   **字卡風格:** 使用現代感字體 (如 Noto Sans TC)，白字黑邊或深色半透明背景，即使在白色網頁上也清晰可見。
*   **轉場:** 使用簡單的淡入淡出 (Fade) 或 快速溶解 (Dissolve)。
*   **Redmine 準備:** 確保在設定頁面已配置好 Redmine URL，並在測試資料中找一個失敗案例準備提交。
*   **數據準備:** 資料庫中準備一個「演示用測試執行 (Demo Run)」，包含約 100 個失敗案例 (混合音頻、多媒體、權限問題)，讓結果看起來更壯觀。

## 💡 額外提示
*   **Redmine 展示:** 如果無法切換到 Redmine 真實介面，確保 GMS 系統內的「成功訊息」夠清楚，例如 *"Ticket #842 created successfully"*。
*   **文字節奏:** 每段文字停留至少 3-4 秒，讓觀眾有時間閱讀。
